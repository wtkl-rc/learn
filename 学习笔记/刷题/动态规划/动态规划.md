# 509.斐波那契数列

2023.10.26大二上

用一个数组去存值，然后把数列算出来存进去，再直接返回就行了

```
class Solution {
    public int fib(int n) {
        List<Integer> list=new ArrayList<Integer>();
        list.add(0);
        list.add(1);
        if(n==0){
            return 0;
        }            
        else if(n==1){
            return 1;
        }         

        for(int num=2;num<=n;num++){
            list.add(list.get(num-1)+list.get(num-2));
        }
        return list.get(n-1)+list.get(n-2);
    }

}
```
