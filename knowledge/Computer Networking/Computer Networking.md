# 计算机网络和因特网

## 1.1什么是因特网

### 1.1.1具体构成描述

这些设备成为主机host或端系统end system（pc，手机）

端系统通过通信链路（communication link）和分组交换机（packet switch）连接到一起

1.1.2服务描述

这些应用涉及多个相互交换数据的端系统，**分布式应用程序** 

*应用程序运行在端系统上，不运行在网络核心中的分组交换机中*
